# TodoApp

Учебный проект на React с управлением задачами.

## Функционал

- Добавление и удаление задач с анимацией
- Фильтрация списка (все/активные/выполненные)
- Сохранение в localStorage
- Валидация поля ввода
- Автофокус и скролл к незавершённым задачам
- Динамические маршруты для деталей задачи
- Оптимизация через useCallback и useMemo

## Технологии

- React (хуки, оптимизация)
- React Router (кастомный Router)
- Context API + кастомные хуки (состояние)
- CSS Modules (стили)
- Feature-sliced Design (структура)
- json-server (mock API)

## Запуск

npm install
npm run dev

# для json-server:
npm run server
